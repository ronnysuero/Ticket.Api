## Backend built With

- [NetCore](https://docs.microsoft.com/en-us/dotnet/) - The webapi framework used

### Server

### Installing Api

Get git source code :

```sh
$ git clone https://github.com/ronnysuero/Ticket.Api
```

Create docker image :

```sh
$ cd Ticket.Api
$ docker-compose build
$ docker-compose up
```

Go to the browser: `http://localhost:5000/` and that's it.
